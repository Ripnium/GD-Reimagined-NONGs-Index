# Read Me
This is the **official** Jukebox index for all NONGs used in GD: Reimagined.
For set up & usage guides, keep reading.

# Setting up
- Boot up *Geometry Dash*
  - Open up the *Geode* menu
- Locate or download the ***Jukebox*** mod
  - If necessary, *restart the game.*
- Go to the *Jukebox* mod.
  
<img width="300" height="345" alt="image" src="https://github.com/user-attachments/assets/41d1b9fb-ce78-4e17-a1ae-2867575ebbf6" />

- Hit the *gear icon at the bottom left.*
<img width="1919" height="1079" alt="image" src="https://github.com/user-attachments/assets/0597aa4e-96a9-4fa8-8ff8-c6f2d8bd4c52" />

- Locate "**Indexes**", and hit "**View**".
<img width="1151" height="197" alt="image" src="https://github.com/user-attachments/assets/864c862d-5649-4f4a-85ec-84f5b6fa81d0" />

- Hit the **+** button
- Copy & Paste this link: `https://raw.githubusercontent.com/Ripnium/GD-Reimagined-NONGs-Index/main/indexes/main.json`
<img width="1919" height="1079" alt="image" src="https://github.com/user-attachments/assets/b99fb2bb-4cf8-434c-83c0-0f24e21ae92e" />

 - Close out of the indexes view, and hit **apply**.
 - Restart the game.

# Using the Index
- After setting up the index, go to a level.
- Next to the song name, you should see a *disc button* (Example level: Checkpoint)
<img width="1205" height="358" alt="image" src="https://github.com/user-attachments/assets/7ec61885-725c-4eb7-b889-7a8cff55d786" />
    
- Click it, then scroll to "Download NONGs"
- GD:R songs have a name like this:
  `[Song Name] (GD:R Cut)`
  (Example level: Are You Ready)
  
<img width="1053" height="207" alt="image" src="https://github.com/user-attachments/assets/1e2cbb97-3a3c-40fb-86ed-8e1423408773" />
  
- Select the song, and you're done.
